This folder contains:
- a VNF package (cirros_vnf.tar.gz) based on a CirrOS image with a single interface
- a NS package (cirros_2vnf_ns.tar.gz) modelling a NS with two CirrOS VNFs interconnected by a virtual link

You can get the CirrOS image required to run this NS in the following link:
http://download.cirros-cloud.net/0.3.4/cirros-0.3.4-x86_64-disk.img

After downloading the image, you will have to add it to your VIM with the name 'cirros034'.
